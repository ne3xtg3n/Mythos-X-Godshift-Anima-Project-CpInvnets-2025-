## **Overall Plot Summary**
In a multi-universal realm, gods of creation and destruction clash, fracturing reality. Prequels show Cassian and Lorcan's rivalry causing Aetheria's near-fall, awakening mortal avatars on Earth. Heavenfall spills divine power to humanity, where Jax (Aether's heir) battles Kael (Oblivion's vessel) in a Godshift—humans ascending amid cosmic war. Episode 2 bonds Jax, Banner, Sarah. Episode 3 introduces new avatars, escalating battles. Themes: Balance, brotherhood, power's cost. Expansions add universal stakes, redemption arcs, human-god fusions.
